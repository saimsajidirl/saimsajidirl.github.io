EDU Manager

This Flutter app allows users to track student data like courses, teachers, and CGPA. It utilizes a Firebase REST API to store and retrieve information.

Word Highlighter

This Flutter app searches the whole phone to find a word anywhere and highlight it. It can search txt, pdf, and many other types of text formats.

To Do List

This to-do list website allows you to store your tasks, making them accessible whenever you need to review, delete, or mark them as complete.

YouTube Videos Downloader

A Tkinter Software that allows users to download YouTube videos in various quality formats. It asks for a link and reads it, then provides the quality to download.

PDF Architect

A Tkinter Software that is a versatile tool enabling you to work with PDFs in a variety of ways. You can edit PDFs, merge PDFs, and convert them into Word format.
